# 64卦指令集说明
 
## 核心理念
- 每条指令对应一个易经卦象，但是不拘泥于卦象的含义可以自我定义。
- 由内核发出 6-bit 信号直接被 FPGA 执行  

## 示例指令
| 卦名        | 功能描述                     | 信号码 (6-bit) |
|------------|-----------------------------|----------------|
| 乾         | 基础计算操作                 | 000001         |
| 坤         | 内存写入                     | 000010         |
| 震         | IO 输出                     | 000011         |
| 巽         | 条件跳转                     | 000100         |
| 兑         | 简单逻辑判断                 | 000101         |
| 离         | 输出显示                     | 000110         |
| 艮         | 保持状态                     | 000111         |
| 坎         | 等待/延时                     | 001000         |

> 注：完整 64 卦指令集待完善，可通过贡献者 PR 补充
